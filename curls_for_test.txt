Примеры запросов
Создание ключа
curl -X POST "http://localhost:8000/key/generate" -H "Content-Type: application/json" -d '{"username": "user"}'

curl -X POST "http://localhost:8000/key/generate" -H "Content-Type: application/json"  -d '{"username": "user1", "rpm_limit": 10, "max_budget": 1000, "budget_duration": "5m"}'

Сохраняет в таблицу LiteLLM_VerificationToken

Генерация нового ключа
curl -X PUT "http://localhost:8000/key/update" -H "Content-Type: application/json" -d '{"key": "sk-xxx", "new_username": "user2"}'

Валидация ключа
curl "http://localhost:8000/key/validate?username=user"



Создать пользователя, Budjet, Rate Limit можно через /user/new или key/create
curl --location 'http://0.0.0.0:4000/user/new' \
--header 'Authorization: Bearer sk-1234' \
--header 'Content-Type: application/json' \
--data '{"user_id": "user2", "rpm_limit": 10, "max_budget": 1000, "budget_duration": "30d"}'


curl --location 'http://0.0.0.0:4000/key/generate' \
--header 'Authorization: Bearer sk-1234' \
--header 'Content-Type: application/json' \
--data '{"user_id": "user3", "rpm_limit": 10, "max_budget": 1000, "budget_duration": "30d"}'
